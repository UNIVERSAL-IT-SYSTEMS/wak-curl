wak-curl
========

Wakanda module to use [cURL](http://curl.haxx.se).

About
-----
**Mac OS X**

* curl/7.39.0 (x86_64-apple-darwin14.0.0) 

* libcurl/7.39.0 
* OpenSSL/1.0.1g 
* zlib/1.2.8 
* libidn/1.29 
* libssh2/1.4.3
 
**Windows**

* curl/7.39.0 (i686-pc-mingw32)  

* libcurl/7.39.0 
* libssh2/1.4.3
